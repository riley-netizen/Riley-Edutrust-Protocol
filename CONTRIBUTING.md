# Contributing to Riley EduTrust Protocol

First off, thank you for considering contributing to Riley Edu Trust! It is people like you who will help bridge the trust gap in the global education industry.

As a protocol initiated by a builder with deep roots in sales in this filed, we value practical solutions that solve real-world transaction flaws. Whether you are a Rust developer, a UI designer, or a study-abroad consultant, your input is welcome.

### 1. Technical Development

### 3. Testing
- Bug reports and UX feedback are extremely valuable. Please open an "Issue" if you find something broken.
